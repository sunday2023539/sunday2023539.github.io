# Personal Portfolio Website

This is a GitHub Pages portfolio template.

## How to use

1. Create a repository named:
   sunday2023539.github.io

2. Upload these files:
   - index.html
   - style.css
   - profile.jpg
   - cv.pdf

3. Replace profile.jpg with your own photo.

4. Replace cv.pdf with your CV file.

5. Edit index.html to change your email, LinkedIn link, projects, and thesis details.

Your website will be available at:
https://sunday2023539.github.io

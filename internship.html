Internship at IGN/LASTIG

During my internship at IGN/LASTIG/ENSG-Géomatique in Paris, France, I worked on the visualization and analysis of ensemble coastal flood data for the Arcachon Basin. My work focused on flood exposure mapping, AOI-based vector deformation, raster polar-log deformation, and 3D web visualization.

Host Organization: IGN / LASTIG / ENSG-Géomatique, Paris, France
Internship Focus: Coastal flood visualization and map deformation
Role: Geovisualization and Spatial Analysis Intern
Tools: Python, ArcGIS Pro, QGIS, Leaflet, MapLibre GL JS

Internship Focus

Flood Exposure Mapping

I processed and visualized ensemble coastal flood data to identify exposed coastal zones around the Arcachon Basin.

AOI-Based Vector Deformation

I calculated AOI centroid coordinates, angles, radius values, and normalized positions between minimum and maximum boundaries.

Raster Polar-Log Deformation

I tested an image-based deformation method by transforming the map into polar-log space using a centre point and distance image.

Selected Outputs

AOI Deformed Polar Space

This output shows each AOI placed in a deformation space using angle and normalized distance between the minimum and maximum boundaries.

Raster Polar-Log Deformation

This output shows an experimental raster-based transformation of the map from normal geographic space into polar-log space.

3D AOI Priority Map

This 3D web visualization uses symbolic height to represent AOI priority. The height does not represent real terrain elevation.

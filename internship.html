Internship at IGN/LASTIG

Coastal Flood Visualization and Map Deformation

During my internship at IGN/LASTIG/ENSG-Géomatique in Paris, France, I worked on the visualization and analysis of ensemble coastal flood data for the Arcachon Basin. My work focused on flood exposure mapping, AOI-based vector deformation, raster polar-log deformation, and 3D web visualization.

Host: IGN / LASTIG / ENSG-Géomatique, Paris, France
Duration: April 2026 – September 2026
Role: Geovisualization and Spatial Analysis Intern

Internship Focus

Flood Exposure Mapping

I processed ensemble coastal flood data and created visual outputs to show flood occurrence and exposed coastal areas around Arcachon Bay.

AOI-Based Vector Deformation

I calculated AOI centroid coordinates, angles, radius values, and normalized positions between minimum and maximum boundaries.

Raster Polar-Log Deformation

I tested an image-based deformation method by transforming the map into polar-log space using a centre point and distance image.

3D Web Visualization

I created a symbolic 3D AOI priority map using MapLibre GL JS, where height represents AOI priority, not real elevation.

Methodology

My workflow started with flood data exploration and AOI selection. I then created minimum and maximum boundary layers to support the deformation framework. For each AOI, I calculated the centroid, angle, radius, and relative position between the boundaries. These values were used to place the AOIs in a deformed polar space.

I also tested raster-based polar-log deformation, where the original map image was transformed into polar space and then modified using a log-based radial transformation. Finally, I developed interactive web visualizations to communicate the outputs more clearly.

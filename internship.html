Internship at IGN/LASTIG

During my internship at IGN/LASTIG/ENSG-Géomatique in Paris, France, I worked on the visualization and analysis of ensemble coastal flood data for the Arcachon Basin.

My work focused on flood exposure mapping, AOI selection, vector deformation, raster polar-log deformation, and interactive 3D web visualization.

Internship Focus

  • Processing and visualizing ensemble coastal flood data
  • Creating flood occurrence and exposure maps
  • Defining Areas of Interest around exposed coastal zones
  • Developing AOI-based vector deformation calculations
  • Testing raster polar-log deformation methods
  • Building interactive web visualizations using Leaflet and MapLibre GL JS

Tools and Technologies

Python, ArcGIS Pro, QGIS, GeoPandas, Shapely, NumPy, Pandas, Matplotlib, Leaflet, MapLibre GL JS.

Selected Outputs

AOI-Based Vector Deformation

I calculated the position of each AOI in polar deformation space using angle, radius, and normalized distance between minimum and maximum boundaries.

Raster Polar-Log Deformation

I tested an image-based deformation method by transforming the map image into polar-log space using a centre point and distance image.

3D AOI Priority Map

I created a symbolic 3D web map using MapLibre GL JS, where AOI height represents priority rather than real elevation.

Reflection

This internship strengthened my skills in geospatial analysis, Python scripting, cartographic visualization, and web mapping. It also improved my ability to explain complex spatial methods clearly and professionally.
